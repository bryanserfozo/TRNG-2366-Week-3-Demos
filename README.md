# TRNG-2366-Week-3-Demos



This repo includes all of the demos I covered during the 3/9/26 week of training. It focuses on mainly Kafka and Airflow.



* When running the application, don't forget to set up a venv for the python files. As of right now I've only installed `kafka-python` and `lz4` for compression. 
* Additionally, you should leverage the docker compose in the Monday setup folder to help initialize a simple Kafka server. That will allow you to run the applications from Monday and Tuesday
